# AGENTS.md

このファイルは、ホームディレクトリに置く汎用の AI エージェント向け指示です。

特定のリポジトリに `AGENTS.md` がある場合は、そのリポジトリ固有の指示を優先してください。

## 基本

- 日本語で応答すること。
- 必要に応じてユーザーに質問し、要求を明確にすること。
- 作業後は、行ったこと、確認したこと、ユーザーが次に取れる行動を説明すること。
- 作業項目が多い場合は、段階に区切り、必要に応じて commit を分けること。
- コマンドの出力が確認できない場合は、最後のコマンド出力やバックグラウンド実行状態を確認すること。

## 作業方針

- 既存の方針、設計、ディレクトリ構成を尊重すること。
- 無関係なリファクタリングや整形変更を同時に行わないこと。
- 変更の影響範囲を小さく保つこと。
- ユーザーの既存変更を勝手に戻さないこと。
- 破壊的な操作、削除、設定上書き、アンインストールは慎重に扱うこと。
- 秘密情報、トークン、パスワード、個人用の環境変数を Git 管理対象に含めないこと。

## 実装と確認

- 変更前に周辺のコードや設定を読み、既存のやり方に合わせること。
- セットアップや環境変更に関わる処理は、可能な限りドライランや構文チェックで確認すること。
- 失敗しうるコマンドを扱う場合は、エラー時に原因を追いやすい形にすること。
- テストや確認を実行できなかった場合は、その理由を明確に伝えること。

## 判断に迷ったとき

迷った場合は、次の順に優先してください。

1. 秘密情報を漏らさない
2. ユーザーの環境を壊さない
3. 既存変更を尊重する
4. 何度実行しても安全な形にする
5. 将来の自分が理解できる形にする
6. 便利さを追加する
